# Fire TV Watcher

Monitor simples para Amazon Fire TV Stick usando ADB.

Funcionalidades:
- Detecta se a TV (entrada do Fire Stick) está ligada ou desligada.
- Detecta mudança de aplicativo em primeiro plano.
- Pode enviar notificações via Telegram (opcional).

Este repositório contém os scripts usados para monitorar o Fire TV a partir de um Raspberry Pi.
